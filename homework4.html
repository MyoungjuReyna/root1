Glossier Website - Repetition



  1. The Glossier website uses many different
    kinds of repetition to engage the viewer
    and create an attractive website


  2. The Glossier font is consistent throughout the site
    giving a cohesive aesthetic.
    Not only is the font used for the websites
    text, but it is also on
    all of the products


























  3. Glossier also uses a complimentary color pallete,
    using the same pinks and whites throughout the
    website. Slight differences in the shades give contrast,
    but the similarity adds to the sites cohesion and repition














  4. The Glossier Website also uses layout repetition
    (spatil relationship) when contrasting between photos
    of models and the photos for the products.

Goop Website - Proximity



  1. The Goop website uses proximity in order to
    organize its different subject sections
    so that the reader can easily navigate to the desired topic


  2. The header of the page is a good example of
    proximity. Below the logo is clear differentiation
    to pages concerning topics like
    "beauty" or "travel".














  3. Lower on the homepage, there is clear differentiation
    between suggested products, ads, and articles.
    Titles on each section as well as slight
    design choices - like black lines around ads -
    create proximity






















  4. There are titles at the beginning of each
    section with suggested/new articles which
    clearly define the topic.
